# InvestraIQ Frontend Console

This is the React-based client interface for the InvestraIQ Multi-Agent Research Terminal.

## Features
* Modern dark-themed glassmorphism interface.
* Responsive historical chart overlays (using Recharts).
* Dynamic event stream progress bar mapping to backend Server-Sent Events (SSE).
* Onboarding tour panel guided by React Joyride.

## Scripts
* Run Dev Server: `npm run dev`
* Run Build Compile: `npm run build`
* Run Linter checks: `npm run lint`
